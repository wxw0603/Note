# 常用命令

| 命令                                      | 功能                     |
| ----------------------------------------- | ------------------------ |
| mount  -t  vboxsf  VMshare  /mnt/VMshare/ | 将所选分区挂载到所选目录 |
| mkdir 目录名                              | 创建目录                 |
| ll                                        | 查看当前目录             |
| make && make install                      | 编译并且安装             |
